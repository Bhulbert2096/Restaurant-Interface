This is a user interface for a restaurants seating availability.  In order to operate it all you need to do is 
press the add a guests button and type in the name of the guest you want to add for the waiting list.  
Once a name has been selected the user needs to select the name they want to add to the seat.  The userneeds to select 
the required seat.  Then the user needs to enter the correct number of guests for the party.  When the guests are 
done and have left the user can then click the release button to give the operator access to the table again.  This will
then allow for the user to grant another guest that table.
